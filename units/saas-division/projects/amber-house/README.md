# Project: Amber House

## 🚀 Description (プロジェクト概要)

**「くらし」の現場をAIで自動化する、ホームサービス事業者向け Vertical SaaS。**

現場作業中で電話に出られない職人さんに代わり、AIが24時間365日の予約受付と顧客管理を完結させます。単なる業務管理ツールではなく、住宅設備データを活用した「次世代のインフラ」を目指します。

### コア機能
- **24/7 AI自動予約**: Web/LINE/音声AIによる予約。
- **設備連動CRM**: 設置機器の型番を管理し、最適な交換時期を提案。
- **決済統合**: 施工完了後の即時決済。
